# Correlation-of-Stock-Market-and-Cryptocurrency-Trading-Volumes-after-Covid19

This project involves aggregation, cleaning and wrangling of trading data from the stock and crypto markets in periods before and after Covid 19 and the investigation of market trends with statistical analysis presented using visualisations revealing positive correlation in the trading volumes in the two markets
